Tockey Server
ルール コマンド 利用規約

基本コマンド

  • /tpa [プレイヤー名] 指定したユーザーにテレポート要求を送信
    /tpa Notch
    • プレイヤー名テレポートするプレイヤーの名前を入力
  • /tpaccept 現在送信されているtpa要求を承認
  • /tpdeny 現在送信されているtpa要求を拒否
  • /clan home クランに設定されているホームにテレポート
  • /ally [join/leave] クランメンバーにしか見えないチャットに参加・退出
    • joinクランチャットに参加
    • leaveクランチャットから退出し、普通のチャットに戻る

Simple TPA

  • /tpa [プレイヤー名] 指定したユーザーにテレポート要求を送信
    /tpa Notch
    • プレイヤー名テレポートするプレイヤーの名前を入力
  • /tpaccept 現在送信されているtpa要求を承認
  • /tpdeny 現在送信されているtpa要求を拒否
  • /tpahere [プレイヤー名] 指定したユーザーを自分にテレポートさせる要求を送信
    /tpahere Notch
    • プレイヤー名テレポートさせるプレイヤーの名前を入力
  • /tpahereall ユーザーを自分にテレポートさせる要求を全員に送信
  • /back テレポート前の場所まで戻る　/clan homeと併用するとバグるので注意
  • /tpauto 届いたテレポート要求の自動承認をトグル
  • /tpatoggle テレポート要求の受信拒否をトグル

情報コマンド

  • t?info 項目１ 項目２ ユーザー、チャンネル、サーバーの情報を表示
    t?info user @taru
    • 項目１オプション (user, channel, server)
    • 項目２（条件）userの場合、対象ユーザーのメンション
  • t?icon 項目１ 項目２ ユーザー、サーバーのアイコンを表示
    t?icon user @taru
    • 項目１オプション (user, server)
    • 項目２（条件）userの場合、対象ユーザーのメンション
  • t?role 項目１ 項目２ ユーザーの所属ロールおよびロール内のユーザーを確認
    t?role check @taru
    • 項目１オプション (member, check)
    • 項目２（条件）memberの場合、対象ユーザー、checkの場合、対象ロール

技術コマンド

  • t?zishin 最新の日本での地震・津波情報を表示 (t?eq, t?earthquakeでも可)
  • t?train 最新の列車遅延状況を表示
  • t?qr 項目１ 文字列をQRコードに変換
    t?qr たるボット
    • 項目１変換する文字列
  • t?readqr QRコード画像の読み取り (画像を添付して実行)
  • t?forex 項目１ 項目２ 外貨と日本円の為替を表示
    t?forex usd 100
    • 項目１変換先の通貨コード
    • 項目２日本円の金額
  • t?ipconfig 項目１ IPアドレス情報を表示
    t?ipconfig 8.8.8.8
    • 項目１IPv4またはIPv6のIPアドレス
  • t?base64 項目１ 項目２ 文字列をBase64に変換 (Base32, Base58にも対応)
    t?base64 encode たるボット
    • 項目１オプション (encode: 符号化, decode: 復号)
    • 項目２変換する文字列
  • t?percent 項目１ 項目２ 文字列をPercentに変換
    t?ppercent encode たるボット
    • 項目１オプション (encode: 符号化, decode: 復号)
    • 項目２変換する文字列

VCコマンド

  • t?play 項目１ YouTube動画音声を再生
    t?play https://www.youtube.com/watch?v=dQw4w9WgXcQ
    • 項目１YouTube動画のリンク
  • t?skip 現在流れている曲をスキップ
  • t?leave ボイスチャットから退出 (強制退出時は再起動まで使用不可)

経済コマンド

  • t?work 働いて報酬を得る
  • t?crime 犯罪を犯して報酬または罰金
  • t?terrorism テロをして大きな報酬または大きな罰則
  • t?balance 現在の所持金を表示
  • t?leaderboard 項目１ サーバー内・全体の所持金ランキングを表示
    t?leaderboard local
    • 項目１オプション (local: サーバー内, global: 全体)
  • t?give 項目１ 項目２ 他ユーザーにお金を送金
    t?give @taru 5000
    • 項目１送金先のメンションまたは名前
    • 項目２送金金額 (allで全額指定可)
  • t?blackjack 項目１ ボットとブラックジャックを遊ぶ
    t?blackjack 5000
    • 項目１賭け金額 (allで全額指定可)

その他

  • t?dice 項目１ 項目２ サイコロを振る
    t?dice 1d100 int
    • 項目１サイコロの面数と回数 (例: 1d100)
    • 項目２（任意）オプション (int: 整数, float: 四捨五入浮動小数点)
  • t?coinflip コインを投げる
  • t?janken ボットとじゃんけんを遊ぶ
  • t?today 今日が何の日かを表示する

ジョークコマンド

  • t?fart VCに参加しておならをする
  • t?kill 項目１ 項目２ ユーザーをキルする
    t?kill @taru thunder
    • 項目１対象ユーザーのメンション
    • 項目２（任意）オプション (shoot, drowned, explode, fall, burn, lava, thunder, magic, froze, attack)
